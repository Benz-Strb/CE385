### Week 1
##### workshop 1.1
สร้างตัวแปร Name, age, branch มาเก็บข้อมูลส่วนตัว แล้วแสดงผลออกมา

##### workshop 1.2
สร้างตัวแปร 5 ตัว ให้เก็บข้อมูลแต่ละประเภท คือ str, num, bool, null, undefined แล้วแสดงผลลัพธ์ Type ของแต่ละตัวแปรออกมา

##### workshop 1.3
สร้างตัวแปรมา 3 ตัว เก็บคะแนนของ 3 วิชา คณิต, วิทยาศาสตร์, ภาษาอังกฤษ ตัวแปร 3 ตัวถัดมาเป็นการคำนวณทางคณิตศาสตร์ total ผลรวมคะแนน, avg ค่าเฉลี่ยของคะแนนวิชาทั้ง 3, comparison เปรียบเทียบคะแนนค่าเฉลี่ยว่า มากกว่าหรือเท่ากับ 80 หรือไม่ และแสดงผลออกมา

##### workshop 1.4
กำหนดตัวแปร username , password, age และเช็คค่าทั้ง 3 ตัว ถ้าตรงทั้ง 3 อนุญาตให้เข้าสู้ระบบได้ ถ้าอายุน้อยกว่า 18 แสดง อายุไม่ถึงเกณฑ์ และไม่ให้เข้าสู่ระบบ ถ้า username, password ไม่ตรง ไม่อนุญาตให้เข้าสู่ระบบเช่นกัน
